# Software Installation Issues

## Problem
User is unable to install software, getting permission errors,
or software is not working after installation.

## Solution

### Step 1 — Check Installation Permission
- Some software requires admin rights
- Contact IT Support to get temporary admin access
- Never install unauthorized software on company devices

### Step 2 — Download from Official Source
- Always download software from IT approved portal
- Go to: itportal.company.com/software
- Find the software you need
- Download and run the installer

### Step 3 — Check Disk Space
- Make sure you have enough disk space
- Go to File Explorer → This PC
- Check available space on C: drive
- Minimum 2GB free space needed for most software

### Step 4 — Disable Antivirus Temporarily
- Sometimes antivirus blocks installation
- Temporarily disable antivirus
- Run the installer again
- Re-enable antivirus after installation

### Step 5 — Run as Administrator
- Right click the installer file
- Select "Run as Administrator"
- Follow installation steps
- Restart laptop after installation

### Step 6 — Contact IT Support
If installation still fails:
- Raise a ticket with IT Support
- Mention software name and version
- Attach screenshot of error message
- IT team will assist within 4 business hours
