# Email Not Working

## Problem
User is unable to send or receive emails, Outlook not loading,
or emails going to spam.

## Solution

### Step 1 — Basic Checks
- Check your internet connection is working
- Make sure Outlook is up to date
- Restart Outlook completely
- Check if email works on web browser at outlook.office.com

### Step 2 — Check Email Storage
- Inbox may be full and blocking new emails
- Delete unnecessary emails from inbox
- Empty deleted items and junk folder
- Storage limit is 50GB per user

### Step 3 — Outlook Not Loading
- Close Outlook completely
- Press Windows + R
- Type: outlook.exe /safe
- Press Enter to open in Safe Mode
- If works in safe mode — add-in issue exists

### Step 4 — Re-add Email Account
- Go to Outlook Settings
- Click Account Settings
- Remove your email account
- Add it back with correct credentials
- Wait for emails to sync

### Step 5 — Check Spam Settings
If emails going to spam:
- Open Junk/Spam folder
- Right click the email
- Select "Not Junk" or "Not Spam"
- Add sender to Safe Senders list

### Step 6 — Contact IT Support
If email still not working:
- Raise a ticket with IT Support
- Mention your email address and error message
- IT team will assist within 2 business hours

## Common Issues
- Cannot send emails → Check SMTP settings
- Cannot receive emails → Check storage limit
- Outlook keeps asking password → Re-add account
- Emails delayed → Server issue, contact IT
