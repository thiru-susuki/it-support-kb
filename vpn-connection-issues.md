# VPN Connection Issues

## Problem
User is unable to connect to company VPN or VPN keeps disconnecting.

## Solution

### Step 1 — Basic Checks
- Check your internet connection is working
- Make sure you are using correct VPN credentials
- Confirm VPN client software is installed correctly

### Step 2 — Restart VPN Client
- Close the VPN application completely
- Wait 30 seconds
- Reopen and try connecting again

### Step 3 — Reinstall VPN Client
If restarting does not work:
- Uninstall the VPN client
- Download latest version from IT portal
- Install and try connecting again

### Step 4 — Check Firewall Settings
- Disable personal firewall temporarily
- Try connecting to VPN
- If it works, add VPN to firewall exceptions

### Step 5 — Contact IT Support
If none of the above works:
- Raise a ticket with IT Support
- Mention your operating system and VPN version
- IT team will assist within 4 business hours

## Common Error Codes
- Error 800 → Network issue, check internet
- Error 691 → Wrong username or password
- Error 619 → Firewall blocking VPN
